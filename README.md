# 🧱🎮 Tetris Game (Python)

## 📌 Overview  
This repository contains a **Tetris Game built using Python programming language**.  
It is a classic block-stacking puzzle game where players arrange falling tetromino shapes to complete lines and score points.

---

## 🎯 Purpose  
- Learn Python game development fundamentals  
- Understand grid-based game logic  
- Practice collision detection and movement handling  
- Build a classic arcade puzzle game  
- Improve problem-solving and spatial thinking  

---

## 🧠 Key Features  
- 🧱 Falling Tetromino Blocks  
- 🎯 Line Clearing System  
- 🏆 Score Tracking  
- ⚡ Increasing Difficulty Over Time  
- 🔄 Game Loop Mechanics  
- 🎮 Keyboard Controls  

---

## 🛠️ Tech Stack  
- 🐍 Python  
- 🎮 Pygame (or similar game library)  
- 💻 VS Code / PyCharm  
- 🧠 Grid & Collision Logic
